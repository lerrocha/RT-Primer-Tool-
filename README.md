This project is based on a computational tool for evaluating reverse transcription (RT) primer specificity and identify potential off-target binding to ribosomal RNA (rRNA). The program analyzes primers designed for the C. elegans gene xbp-1 and ets-4 and evaluates their binding similarity to both target gene sequences and rRNA sequences.
 Calculates reverse complement of primers

 Scans sequences to determine best binding location

 Measures percent match between primer and target

 Evaluates off-target rRNA binding risk

Generates single-base mutant primers

Suggests improved primers with reduced rRNA binding
